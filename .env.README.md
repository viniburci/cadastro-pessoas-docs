# Configuração de Variáveis de Ambiente

## Instruções de Uso

1. Copie o arquivo `.env.example` para `.env`:
   ```bash
   cp .env.example .env
   ```

2. Edite o arquivo `.env` com suas configurações locais:
   - `SPRING_PROFILES_ACTIVE`: escolha o perfil (vinicius ou doban)
   - `DOBAN_EXCEL_FOLDER`: caminho para o arquivo Excel

3. O arquivo `.env` está no `.gitignore` e não será commitado

## Variáveis Disponíveis

- `SPRING_PROFILES_ACTIVE`: Perfil do Spring (vinicius ou doban)
- `SPRING_DATASOURCE_URL`: URL do banco de dados
- `SPRING_DATASOURCE_USERNAME`: Usuário do banco
- `SPRING_DATASOURCE_PASSWORD`: Senha do banco
- `DOBAN_EXCEL_FOLDER`: Caminho para o arquivo Excel
- `SERVER_PORT`: Porta do servidor (padrão: 8080)

## Nota

Cada desenvolvedor deve ter seu próprio arquivo `.env` com suas configurações locais.
O arquivo `.env.example` serve como template.
